# ActiveBox-landing-page
A modern responsive landing page built with HTML and CSS.
## Description
ActiveBox is a modern and responsive one-page landing website built using HTML5 and CSS3.  
The project focuses on clean layout structure, responsive design, and professional UI styling.
## Technologies Used
- HTML5
- CSS3
- Flexbox
- Google Fonts
- Font Awesome
## Features
- Responsive design for different screen sizes
- Modern and clean layout
- Navigation bar
- Hero section
- Feature cards section
- Social media icons
- Smooth styling and spacing
## What I Learned
- How to structure a professional landing page
- How to use Flexbox for layout alignment
- How to make a website responsive
- How to organize CSS properly
- How to use external resources like Google Fonts and Font Awesome
## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
